# Secure Login System

## Project Description

This project is a secure login web application developed using Python Flask.

The application allows users to:

- Register
- Login
- Logout
- Store passwords securely using bcrypt hashing
- Protect against SQL Injection using parameterized queries
- Manage user sessions

---

## Features

- User Registration
- Secure Password Hashing (bcrypt)
- Login Authentication
- SQL Injection Protection
- Session Management
- Logout Feature

---

- Flask

## Installation

Install Python.

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open browser

```
http://127.0.0.1:5000
```
